Password Manager

Technologies Used:
  • Python
  • SQL
  • PyQt5
  • XML
  • Tkinter
  • Database hosted locally
Description:

This project was developed using old Python libraries to create a simple front end of a password manager, developed in the summer off from school in an attempt to further my knowledge in Python.


The application displays a table of information entered by the user in order to help them remember their passwords and give them a location where they can find all the usernames and passwords they create, since there could be a lot of them.


Add a Password

Users are given a prompt as shown above to enter the service, username, and password. Once they refresh, they will see it in the table.


Search

Users can search for a password if they have an extensive table by inputting the service. The result will be a pop up of all instances of that services in the table, or a pop up saying no results were found.


Delete

Users can delete a password by entering the index of the password in the table. When they refresh, that password will be deleted.
